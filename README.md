# Financial-Data-Analysis

## 📌 Objective
This project analyzes financial data to evaluate budget performance, cost distribution, and factors contributing to business losses.

## 📊 Tools Used

-Excel / Power BI
-Data Cleaning
-Financial  Analysis
-Data Visualization

## 🧹 Data Preparation

- Cleaned dataset by removing duplicates and handling misiing values
- Standardized financial categories (Budget, Expenses, Profit/Loss)
- Created calculated fields (Profit %, Total profit/loss)
- Aggregated data by category, month, and transaction level
- Verified data consistency for accurate financial analysis

## 📈 Key Insights

### 1. Overall Financial Performance
- Total Budget: 1M  
- Actual Expense: 1.33M  
- Total Profit/Loss: -95K  
- Avg Profit %: -42.8%  
👉 Company is operating at a loss due to overspending

### 2. Budget vs Expense Gap
- Actual expenses exceed budget across multiple categories  
👉 Poor budget control is a major issue

### 3. Category-Level Analysis
- High expenses observed in categories like Salaries, Rent, and Operations  
👉 Fixed costs are significantly impacting profitability

### 4. Profit/Loss Distribution
- Majority transactions result in loss (~54%)  
👉 Business model is not generating consistent profit

### 5. Monthly Trend
- Expenses consistently higher than budget across months  
👉 Indicates lack of financial planning and control

### 6. Transaction Analysis
- Frequent loss-making transactions observed  
👉 Inefficiencies in cost management or pricing strategy

## 💡 Business Recommendations

- Implement strict budget monitoring to control overspending  
- Reduce high fixed costs such as rent and operational expenses  
- Optimize salary structure or workforce allocation  
- Identify and eliminate loss-making transactions  
- Improve pricing or revenue strategies to increase profitability  
- Conduct detailed cost analysis for each category
